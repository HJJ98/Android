## Login 
这个文件中是Android studio开发的用户登录页面，其实很简单，这里主要是利用LoginDailogFragment实现的弹窗登录，点击用户名个密码输入框然后弹窗进行输入，在新的窗口进行输入之后，输入的内容，会返回到之前的登录页面，然后再点击登录，即可登录了。

很漂亮的一个前端页面
